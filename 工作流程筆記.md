# Claude Code 工作流程筆記

---

## 開始工作前

### 繼續舊專案
1. 點桌面的 Claude Code 捷徑（會自動回到上次的資料夾）
2. 開新對話，說「請先讀 `handoff.md`」
3. 直接開始工作

### 切換到其他專案
1. Claude Code → 左上角 `≡` → `File` → `Open File`
2. 選取該專案的資料夾 → 開啟
3. 開新對話，說「請先讀 `handoff.md`」

### 開全新專案
1. 在桌面建立新資料夾
2. Claude Code → `≡` → `File` → `Open File` → 選取新資料夾
3. 告訴 Claude 要做什麼，Claude 會自動建立 `handoff.md`

---

## 結束工作時

說「**存檔**」或「**收工**」→ Claude 自動更新 `handoff.md`

---

## 重要檔案說明

| 檔案 | 位置 | 用途 |
|------|------|------|
| `handoff.md` | 各專案資料夾內 | 專案交接文件，記錄現況和待辦 |
| `CLAUDE.md` | `C:\Users\Chi\.claude\` | 全域設定，所有專案都適用 |

---

## 跨裝置或搬移資料夾

1. 說「**存檔**」確保 `handoff.md` 是最新的
2. 複製整個專案資料夾到新位置／新電腦
3. 新電腦建立全域 `C:\Users\Chi\.claude\CLAUDE.md`（內容同現在這台）
4. 用 Claude Code 開啟資料夾
5. 說「請先讀 `handoff.md`」繼續工作
